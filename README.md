# Freshcode Website Course

My solution to last task on Freshcode course about how to make websites with HTML and CSS. I take this course to close my pracice in University

In my solution I connect TailwindCSS for reset.css and for easier development, because for me easier to build up design via emmet and a bunch of classes. In the end I rewrite all tailwind to default CSS, so in production version it used only for reset right now

About figma design - It has a lot of errors, as example some sizes is 61, 57 or 97 or even values with floating point. Also design don't have a list of used colors with names as well as fonts.

I like to make websites pixel-perfect, but in this case it will take too much effort in development. If it will be on work or in production team, it will better to first discuss errors with design team, so they can fix them and after make pixel-perfect website

You can run this project by cloning it, install dependencies via pnpm and run

Website was also deployed on Vercel - https://uni-practice-freshcode-website-cour.vercel.app

Code was checked by w3 validator
